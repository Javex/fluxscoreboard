FluxFingers Scoreboard Documentation
====================================

The FluxFingers Scoreboard is a web application for the hack.lu CTFs. It is
built using the `Pyramid`_ web application framework and uses several
components including the `SQLAlchemy`_ ORM for easier and more consistent
coding.

.. _Pyramid: http://docs.pylonsproject.org/en/latest/docs/pyramid.html 
.. _SQLAlchemy: http://www.sqlalchemy.org/
